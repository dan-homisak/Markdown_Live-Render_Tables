# Repository Instructions

- Before finishing any code change, run the project type check or compile command and resolve all reported errors.
- After every prompt that changes this repository, run `./Build_and_Install` before responding so the latest extension build is installed into VS Code.
- Do not leave known red-highlight VS Code Problems diagnostics in touched files. If a diagnostic cannot be reproduced from the terminal, inspect and resolve it in the editor context before considering the change complete.
